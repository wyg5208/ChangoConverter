   https://github.com/wyg5208/ChangoConverter.git

第二步：推送代码到GitHub（3分钟）
============================================================

方式A：使用自动化脚本（推荐）✨
───────────────────────────────────────────────

1. 双击运行：github_publish.bat

2. 选择：[1] 初始化Git仓库
   - 输入用户名：wyg5208
   - 输入邮箱：wyg5208@126.com
   - 输入仓库URL：https://github.com/wyg5208/ChangoConverter.git

3. 选择：[5] 完整发布流程
   - 输入提交信息：Initial commit: ChangoConverter v1.4.1
   - 输入标签名称：v1.4.1

4. 等待完成！✅


方式B：手动命令
───────────────────────────────────────────────

打开PowerShell，运行以下命令：

# 1. 初始化
git init
git config user.name "wyg5208"
git config user.email "wyg5208@126.com"

# 2. 添加远程仓库
git remote add origin https://github.com/wyg5208/ChangoConverter.git

# 3. 提交代码
git add .
git commit -m "🎉 Initial commit: ChangoConverter v1.4.1"

# 4. 推送到GitHub
git branch -M main
git push -u origin main

# 5. 创建标签
git tag -a v1.4.1 -m "Release v1.4.1"
git push origin v1.4.1

============================================================
第三步：创建Release发布（5分钟）
============================================================

1. 【打包程序】（如果还没打包）
   双击运行：快速打包.bat
   生成文件：dist\ChangoConverter.exe

2. 【创建压缩包】
   压缩以下内容为 ChangoConverter_v1.4.1_Windows_x64.zip：
   - dist\ChangoConverter.exe
   - README.md
   - resources\icon.ico

3. 【访问Release页面】
   https://github.com/wyg5208/ChangoConverter/releases/new

4. 【填写Release信息】
   - 选择标签：v1.4.1
   - 标题：ChangoConverter v1.4.1 - 专业图标 + 目录优化
   - 描述：复制 docs\GitHub发布指南.md 第4.4节的内容

5. 【上传文件】
   拖拽：ChangoConverter_v1.4.1_Windows_x64.zip

6. 【发布】
   点击 "Publish release"
