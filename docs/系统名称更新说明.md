1. **技术依赖**: 本软件基于Pandoc引擎构建
2. **技术透明**: 用户需要知道底层使用的技术
3. **文档引用**: 技术文档和链接需要保持准确
4. **库名称**: `pypandoc`是Python包名，不能更改
