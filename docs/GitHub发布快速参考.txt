⚠️ 常见问题
────────────────────────────────────────────────────
Q: 推送时要求输入密码？
A: 使用 Personal Access Token
   Settings → Developer settings → Personal access tokens
   创建token，复制并使用它代替密码

Q: 推送被拒绝？
A: 先拉取：git pull origin main --rebase
   再推送：git push

Q: 文件太大无法上传？
A: GitHub限制100MB，不提交大文件
   或使用 Git LFS

Q: 如何删除远程分支？
A: git push origin --delete 分支名

Q: 如何修改上次提交信息？
A: git commit --amend -m "新的提交信息"
   git push --force
