📁 文件说明
-----------

1. 快速打包工具
   • 快速打包.bat           - 一键打包菜单（推荐使用）
   • build_exe.bat          - 打包为 EXE 文件
   • build_msi.bat          - 打包为 MSI 安装包
   • prepare_release.bat    - 准备发布压缩包

2. 配置文件
   • setup.py               - MSI 打包配置
   • build_config.py        - PyInstaller 配置
   • requirements_build.txt - 打包依赖列表

3. 文档
   • BUILD_GUIDE.md         - 详细打包指南（必读）
   • 打包说明.txt           - 本文件


🚀 快速开始
-----------

方式一：使用一键打包（最简单）
   1. 双击运行 "快速打包.bat"
   2. 选择 [1] 快速打包
   3. 等待完成，在 dist\ 文件夹找到 PandocConverter.exe

方式二：手动打包 EXE
   1. 激活虚拟环境：venv\Scripts\activate
   2. 安装依赖：pip install pyinstaller
   3. 运行：build_exe.bat
   4. 输出：dist\PandocConverter.exe

方式三：手动打包 MSI
   1. 激活虚拟环境：venv\Scripts\activate
   2. 安装依赖：pip install cx-Freeze
   3. 运行：build_msi.bat
   4. 输出：dist\*.msi


📦 打包流程
-----------

完整打包流程：
   第一步：打包 EXE 文件
      运行：build_exe.bat
      生成：dist\PandocConverter.exe

   第二步（可选）：打包 MSI 安装包
      运行：build_msi.bat
      生成：dist\PandocConverter-1.0.0-win64.msi

   第三步：准备发布包
      运行：prepare_release.bat
      生成：PandocConverter-v1.0.0.zip


📋 环境要求
-----------

必需：
   • Python 3.7+
   • pypandoc
   • pyinstaller（打包 EXE）
   • cx-Freeze（打包 MSI）

可选：
   • icon.ico（自定义图标）
   • NSIS（高级安装程序）


⚠️ 常见问题
------------

Q: 打包失败怎么办？
A: 1. 确保虚拟环境已激活
   2. 检查是否安装了 pyinstaller
   3. 查看错误日志，可能缺少依赖

Q: 打包的程序无法运行？
A: 1. 在其他电脑测试
   2. 查看是否有杀毒软件拦截
   3. 使用 --debug 模式重新打包

Q: EXE 体积太大？
A: 1. 使用 --upx 压缩
   2. 排除不必要的模块
   3. 考虑使用目录模式而非单文件

Q: 用户反馈 Pandoc 未安装？
A: 这是正常的，打包的程序不包含 Pandoc
   用户需要单独安装：https://pandoc.org/installing.html
