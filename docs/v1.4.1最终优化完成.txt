【已知问题】
================================================================

1. PyPandoc库需要手动安装
   pip install pypandoc

2. PDF转换警告（正常，可忽略）
   - 管理员权限警告
   - 字体缺失警告

3. DOCX字体设置需要python-docx
   pip install python-docx

【使用建议】
================================================================

日常使用:
1. 直接双击运行（不要以管理员身份）
2. 选择转换模式（单文件/批量）
3. 选择输入和输出格式
4. 点击深绿色"开始转换"按钮
5. 右侧查看实时转换日志

PDF转换:
1. 系统自动处理Emoji
2. 自动使用XeLaTeX引擎
3. 自动设置中文字体
4. 无需手动优化

批量转换:
1. 选择输入格式（6选1）
2. 系统只扫描该格式文件
3. 大幅提升扫描效率
4. 转换结果保存到源文件夹
