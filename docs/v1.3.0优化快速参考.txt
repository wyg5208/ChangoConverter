【高级选项对话框】
===================================================================

Pandoc转换选项：
├─ ☑ 生成独立文档 (--standalone)
│  └─ 适用于HTML、LaTeX等需要完整文档结构的格式
├─ ☐ 生成目录 (--toc)
│  └─ 在文档开头自动生成目录
└─ ☐ 章节自动编号 (--number-sections)
   └─ 为章节添加序号

DOCX文档字体设置：
├─ 字体: Times New Roman / Arial / Calibri / 默认
└─ 字号: 10 / 11 / 12 / 14 / 16 / 18 / 默认

对话框按钮：
├─ [确定] - 保存设置并关闭
└─ [恢复默认] - 重置所有选项为默认值

【常见问题】
===================================================================

Q1: TAB页面的常用格式和其他格式有什么区别？
A1: 常用格式TAB包含最常用的12种格式，其他格式TAB包含所有
    其余格式。可以根据需要在两个TAB之间切换。

Q2: 高级选项按钮在哪里？
A2: 在输出格式选择区域下方，显示为 ⚙️ 高级选项 按钮。
    点击后会弹出独立的配置对话框。

Q3: 日志区域可以调整大小吗？
A3: 可以。日志区域支持窗口缩放，可以通过拖动窗口边缘来
    调整整体窗口大小，日志区域会自动适应。

Q4: 如何恢复高级选项的默认值？
A4: 打开高级选项对话框，点击左下角的 [恢复默认] 按钮，
    所有选项会恢复为默认值。

Q5: 文件覆盖对话框显示不全怎么办？
A5: v1.3.0已经优化了对话框尺寸（600x400），应该能完整
    显示所有内容。如果还有问题，请检查显示器分辨率。

【技术细节】
===================================================================

├─ 原生Tkinter TAB控件
├─ 自动处理TAB切换

高级选项对话框: tk.Toplevel
├─ 独立窗口（650x350）
├─ 模态对话框
└─ 自动居中显示

日志区域: tk.Text
├─ 高度: 20行（从8行增加）
├─ 字体: Consolas 9pt
└─ 自动滚动条

文件覆盖对话框: tk.Toplevel
├─ 尺寸: 600x400（从500x280增加）
├─ 模态对话框
└─ 自动居中显示
